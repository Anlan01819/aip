# AI项目管理平台

基于设计文档和图片资源搭建的AI项目管理平台页面框架。

## 项目结构

```
src/
├── components/
│   ├── Auth/                 # 认证相关组件
│   ├── Dashboard/            # 仪表板组件
│   ├── Layout/               # 布局组件
│   ├── Organization/         # 组织管理组件
│   └── Project/              # 项目管理组件
├── contexts/                 # React Context
├── lib/                      # 工具库和配置
└── App.tsx                   # 主应用组件
```

## 页面框架

### 1. 主布局 (MainLayout)
- 顶部导航栏：包含应用标题、导航标签、用户信息
- 响应式设计：支持移动端侧边栏
- 导航菜单：我的组织、探索组织

### 2. 我的组织 (OrganizationList)
- 显示用户所属的组织列表
- 组织卡片：包含项目统计、最新项目
- 创建组织功能
- 进入工作台按钮

### 3. 探索组织 (OrganizationExplore)
- 浏览所有公开组织
- 组织信息展示
- 加入组织功能
- 创建组织入口

### 4. 项目列表 (ProjectList)
- 组织内部项目展示
- 项目状态标签
- 项目详情查看
- 左侧边栏任务提示

### 5. 仪表板 (Dashboard)
- 用户欢迎信息
- 快速操作按钮
- 我的任务列表
- 我的项目网格
- AI聊天功能

## 技术栈

- **前端框架**: React 18 + TypeScript
- **路由**: React Router DOM
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **日期处理**: date-fns
- **后端**: Supabase (PostgreSQL + 实时订阅 + 认证)

## 开发说明

### 环境变量
创建 `.env` 文件：
```bash
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 启动开发服务器
```bash
npm install
npm run dev
```

### 数据库设置
1. 在Supabase控制台创建项目
2. 执行 `database/setup_rls.sql` 启用安全策略
3. 执行 `database/fix_existing_users.sql` 设置触发器

## 设计原则

1. **UTF-8编码**: 所有文件使用UTF-8编码
2. **组件化**: 高度模块化的组件设计

5. **可访问性**: 遵循Web可访问性标准
